A simple and creative iOS application built with Swift. 

Key Features:
Musical Notes - Users can tap different buttons to play specific sound files (.wav).
Visual Feedback - When a button is pressed, its opacity decreases briefly, giving the user a clear visual cue.
Audio Integration - Uses the AVFoundation framework to deliver high-quality sound playback.

Logic:
When a user taps a button, the app identifies its name and plays the corresponding sound file. It also uses a quick fade-out animation to show which button was pressed.

Tech Details:
Language - Swift
Framework - UIKit & AVFoundation
Architecture - MVC (Model-View-Controller)
